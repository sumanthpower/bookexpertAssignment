# Project Overview

The Employee Management Dashboard is a React-based web application for managing employee records efficiently. It provides a user-friendly interface to add, edit, delete, and view employee details with real-time search and filtering. The dashboard also supports image uploads, status toggling, and printing employee data.

# Key Features

- Add Employee
- Enter Full Name, Gender, Date of Birth, State, and Profile Image.
- Set Active / Inactive status.
- Save employee to the dashboard table.

# Search Employees

- Search by Employee ID or Full Name.
- Works together with filters.

# Filter Employees

- Filter by Gender (Male / Female / All).
- Filter by Status (Active / Inactive / All).

# Toggle Status

- Change employee Active / Inactive status directly in the table.

- Print Employee Table
- Print the employee list from the dashboard.
- Sticky Layout
